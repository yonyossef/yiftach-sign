# שלט הכוונה - קיבוץ יפתח

מערכת שלט הכוונה לעסקים בקיבוץ יפתח עם דף נחיתה אינטראקטיבי ופאנל ניהול.

## תכונות

- דף נחיתה עם הדמיה של השלט
- לוחות אינטראקטיביים עם קישורים לגוגל מפות
- פאנל ניהול לעריכת הלוחות
- אימות משתמשים
- אחסון נתונים בקובץ JSON

## התקנה

```bash
npm install
```

## הפעלה

```bash
npm start
```

השרת יעלה על `http://localhost:3000`

## משתמש מנהל

- שם משתמש: `admin`
- סיסמה: `admin123`

**חשוב:** שנה את הסיסמה ב-`config.json` לפני הפעלה בייצור!

## דפים

- דף נחיתה: `/`
- התחברות מנהל: `/admin.html`
- פאנל ניהול: `/admin-panel.html`

## פריסה ל-Railway

הפרויקט מוכן לפריסה ב-Railway. פשוט חבר את ה-repository ל-Railway והוא יזהה את הקובץ `railway.json` ויפרוס אוטומטית.

### משתני סביבה ב-Railway

- `PORT` - נקבע אוטומטית על ידי Railway
- `SESSION_SECRET` - מומלץ להגדיר סוד חזק בייצור
- `NODE_ENV` - הגדר ל-`production` בייצור

## מבנה הפרויקט

```
t1/
├── server.js              # Express server
├── data.json              # נתוני הלוחות
├── config.json            # הגדרות משתמש (לא נשמר ב-git)
├── public/                # קבצים סטטיים
│   ├── index.html         # דף נחיתה
│   ├── admin.html         # דף התחברות
│   ├── admin-panel.html   # פאנל ניהול
│   └── admin/             # סקריפטים של פאנל ניהול
└── package.json           # תלויות
```
